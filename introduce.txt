AI 기반 데이터 분석 및 자동 시각화 시스템
=========================================

프로젝트 소개
-----------

이 프로젝트는 CSV 형태의 데이터를 자동으로 분석하고, 
인공지능(LLM)을 활용하여 최적의 시각화를 자동 생성하는 
웹 기반 데이터 분석 플랫폼입니다.

데이터 분석가나 개발자가 별도로 시각화 코드를 작성하지 않아도, 
AI가 데이터의 특성을 이해하고 가장 적합한 그래프와 차트를 
자동으로 생성해줍니다.


핵심 기능
--------

📊 자동 데이터 분석
  - CSV 파일을 업로드하면 자동으로 통계 분석 수행
  - 평균, 중앙값, 표준편차, 상관관계 등 주요 통계치 계산
  - 데이터 패턴과 이상치 자동 감지

🤖 AI 기반 시각화 자동 생성
  - GPT-4 등 최신 LLM 모델을 활용한 지능형 시각화
  - 데이터의 특성에 맞는 최적의 그래프 타입 자동 선택
  - Python 시각화 코드를 AI가 직접 생성하고 실행

🌐 웹 기반 인터페이스
  - 브라우저에서 간편하게 사용 가능
  - 파일 선택과 모델 선택만으로 분석 시작
  - 생성된 시각화를 실시간으로 웹에서 확인

☁️ 클라우드 네이티브
  - Docker 컨테이너 기반 배포
  - Kubernetes를 통한 확장 가능한 운영
  - CI/CD 파이프라인 지원


작동 원리
--------

1. 사용자가 CSV 데이터 파일을 선택
2. 시스템이 데이터를 자동으로 분석하여 주요 통계 추출
3. 분석 결과를 LLM에게 전달
4. LLM이 데이터 특성에 맞는 시각화 Python 코드 생성
5. 생성된 코드를 실행하여 그래프 이미지 생성
6. 웹 브라우저에 결과 표시


지원하는 데이터 타입
-----------------

현재 다음과 같은 데이터셋을 분석할 수 있습니다:

• 보험 데이터 (Insurance Data)
  - 나이, BMI, 보험료, 흡연 여부, 지역 등
  - 보험료 영향 요인 분석
  - 연도별 트렌드 비교 (2022년, 2023년)

• 리뷰 데이터 (Review Data)
  - 평점, 리뷰 텍스트, 사용자 정보
  - 감성 분석 및 평점 분포 분석
  - 리뷰 길이와 평점 상관관계


사용 가능한 AI 모델
-----------------

• OpenAI 모델
  - GPT-4 (최고 품질)
  - GPT-3.5-turbo (빠른 응답)

• Ollama 로컬 모델
  - Llama 3.2 (오픈소스)
  - Qwen 2.5 (다국어 지원)


주요 활용 사례
------------

📈 비즈니스 인텔리전스
  - 판매 데이터 자동 분석 및 시각화
  - 고객 행동 패턴 분석
  - 시장 트렌드 파악

🏥 헬스케어 데이터 분석
  - 의료 보험 데이터 분석
  - 환자 데이터 패턴 인식
  - 건강 지표 상관관계 분석

💬 고객 피드백 분석
  - 리뷰 및 설문 데이터 시각화
  - 고객 만족도 추이 분석
  - 긍정/부정 피드백 비율 분석

📊 학술 연구
  - 실험 데이터 시각화
  - 통계 분석 자동화
  - 연구 결과 발표 자료 생성

기술적 특징
----------

• 클린 아키텍처 (Clean Architecture)
  계층형 구조로 설계되어 유지보수가 쉽고 확장 가능합니다.

• 디자인 패턴 활용
  Factory Pattern, Strategy Pattern 등을 적용하여
  유연하고 재사용 가능한 코드를 작성했습니다.

• 타입 안정성
  Python 타입 힌트와 MyPy를 활용하여 런타임 오류를 최소화했습니다.

• 비동기 처리
  FastAPI와 asyncio를 활용하여 높은 동시성을 지원합니다.
